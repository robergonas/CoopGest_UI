Cambiar contraseña

El nombre de usuario es obligatorio. El nombre de usuario debe tener al menos 4 caracteres.
La contraseña es obligatoria. La contraseña debe tener al menos 8 caracteres.
La nueva contraseña es obligatoria. El contraseña debe tener al menos 8 caracteres.
La confirmación de la contraseña es obligatoria. Las contraseñas no coinciden.

Tu contraseña debe cumplir con los siguientes requisitos:
Longitud minima de ocho caracteres
Almenos una mayúscula
Almenos un número
